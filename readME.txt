TravelPoint - Платформа для путешественников

React
Node.js
Express.js
SQLite

Веб-платформа для публикации и обсуждения статей о путешествиях.

Что есть на сайте:

Аутентификация с JWT токенами
CRUD операции для статей и комментариев
Админ-панель с управлением контентом
Система лайков статей
Фильтрация по категориям
Пагинация
Адаптивный дизайн

Предварительные требования
Node.js 16+ 
npm

Установка и запуск

1. Клонируйте репозиторий
git clone https://github.com/salv1kVM/TravelPoint.git
cd TravelPoint

2. Запустите в VScode 2 терминала

3. Запустите бэкенд
cd ../backend
npm install
npm run dev

4. Запустите фронтенд 
cd ../frontend
npm install
npm run dev

5. Перейти по хостингу во фронтенде
